## Clasification
We have 
- **Obsidian tools**: which are tools that interact with the user's vault. The files of this tools are inside the *obsidian* folder and have the prefix *obs*.
- **Other tools**: Can be native Google tools or other kind.

## Special characteristics in tools
- obsEdit -> Edit an existing note -> Diff, accept or decline changes
- obsCreate -> Creates a note -> Content generation with Gemini
- obsRead -> Reads a note -> Image parsing
